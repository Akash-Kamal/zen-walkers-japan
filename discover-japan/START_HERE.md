## 📍 What You Need To Do

### Three Things Only:

**1. Create EmailJS Account** (2 minutes)
   - Visit https://www.emailjs.com/
   - Sign up (free)
   - Verify email

**2. Get Your Credentials** (3 minutes)
   - Public Key → Account → API Keys
   - Service ID → Email Services
   - Template ID → Email Templates

**3. Update Code** (2 minutes)
   - Open `src/components/ContactModal.tsx`
   - Replace 3 placeholder values
   - Save

**4. Install Package** (1 minute)
   ```bash
   npm install @emailjs/browser
   ```

**5. Test** (2 minutes)
   ```bash
   npm run dev
   ```
   - Click "Send Message"
   - Fill form
   - Check email
